# lower-electricity-bills-using-blockchain
Lower Electricity Bills using Blockchain 

# Softwares
1. Ganache
2. VS Code
3. Visual Studio with C++ Build Tools
4. NodeJs
5. Python3
6. git

# Packages
1. npm install -g truffle
2. pip install web3

# Process
1. Install Softwares
2. Install Packages
3. Clone Repo
4. Launch Ganache
5. Migrate Smart Contracts onto Ganache
6. Replace the smart contract address in src/ca.py
7. python app.py
